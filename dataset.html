  • About

    This dataset includes cases generally associated with a place or thing (for example parks, streets, or buildings) and created July 1, 2008 or later. Cases generally logged by a user regarding their own needs (for example, property or business tax questions, parking permit requests) are not included. See the Program Link and detailed attachment below for more information..

  • Column Description

    1. Call Type Group: Call types are divided into four main groups: Fire, Alarm, Potential Life Threatening and Non Life Threatening.
    2. Call Type: Call types of 311 case.
    3. Station Area: Fire Station First Response Area associated with the address of the incident.
    4. Battalion:Emergency Response District (There are 9 Fire Emergency Response Districts.
    5. Closed: The date and time when the service request was closed
    6. Status: A single-word indicator of the current state of the service request. (Note: GeoReport V2 only permits ‘open’ and ‘closed’)
    7. Supervisor District: San Francisco Supervisor District as defined in 'Supervisor Districts as of April 2012'.
    8. Latitude: Latitude of the location, using the WGS84 projection.
    9. Longitude: Longitude of the location, using the WGS84 projection.
    10. Media URL:A URL to media associated with the request, e.g. an image.
